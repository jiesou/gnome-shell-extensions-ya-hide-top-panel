# Yet Another Hide Top Panel

A GNOME Shell extension that hides the top panel bar. But keeps it in overview, with hot corners always available. Without Autohide/Intellihide feature.

Basically the same as `Panel Visibility OFF` + `Panel Visibility in Overview ON` in [jrahmatzadeh/Just Perfection](https://extensions.gnome.org/extension/3843/just-perfection).

It may not work well with [Dash to Dock](https://extensions.gnome.org/extension/307/dash-to-dock) or Ubuntu Desktop, try disabling Panel Mode or changing the Panel Position.

## Why created this

- [sonersg/Hide Topbar](https://extensions.gnome.org/extension/9031/hide-topbar) flickers when entering overview.
- [tuxor1337/Hide Top Bar](https://extensions.gnome.org/extension/545/hide-top-bar) makes hot corners feel weird and harder to trigger.
- [jrahmatzadeh/Just Perfection](https://extensions.gnome.org/extension/3843/just-perfection) flickers after unlocking shell.
